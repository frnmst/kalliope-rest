Developer Interface¶

Main Interface¶

kalliope_rest.get_kalliope_version(base_uri, username, password)¶

Return Kalliope’s version.

kalliope_rest.get_synapses(base_uri, username, password)¶

Returns a list of all available synapses and their details.

kalliope_rest.get_synapse(base_uri, username, password, synapse_name)¶

Returns the selected synapse and its details.

kalliope_rest.get_listening_status(base_uri, username, password)¶

Tells if Kalliope is ready to listen for vocal orders.

kalliope_rest.execute_by_name(base_uri, username, password, synapse_name, voice)¶

Executes a synapse with the specified name.

kalliope_rest.execute_by_order(base_uri, username, password, order, voice)¶

Execute the specified order by text.

kalliope_rest.execute_by_audio(base_uri, username, password, audio_file, voice)¶

Execute the specified order by audio.

Exceptions¶

exception kalliope_rest.AudioFileError¶

Raises an exception if the audio file is not found or if there is a filesystem error.

exception kalliope_rest.AudioFileFormatError¶

Raises an exception if the provided audio file is not conforming to the following format specifications (mime types): ‘audio/wav’, ‘audio/x-wav’, ‘audio/mpeg3’, ‘audio/x-mpeg-3’
