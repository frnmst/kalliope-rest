CLI Usage helps¶

Main help¶

usage: kalliope_rest [-h] [-v] {kv,sps,sp,listening,exec} …

Kalliope REST API frontend

positional arguments:
{kv,sps,sp,listening,exec}
kv Get the version of Kalliope sps Get all the synapses sp Get the selected synapse listening tells if Kalliope is waiting for orders exec Execute a synapse by different criterias
optional arguments:
-h, --help show this help message and exit
-v, --version show program’s version number and exit

Return values: 0 OK, 1 API Error, 2 Invalid command
