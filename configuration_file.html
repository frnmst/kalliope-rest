Configuration File¶

Upon the first run of the CLI script, you will find the configuration file under

~/.config/kalliope_rest/kalliope_rest.conf

What follows is an example of configuration file which is included in this repository

[Network]
Host = 127.0.0.1
Port = 5000

[Administration]
Username = admin
Password = secret

kalliope-rest
