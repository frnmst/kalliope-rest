Testing
=======

Before running the tests you need to install the library's dependencies and 
a couple more:

- requests-mock_
- pyfakefs_

.. _requests-mock: https://requests-mock.readthedocs.io/en/latest/

.. _pyfakefs: http://pyfakefs.org

Now, run the following command in a terminal,

::

    $ python setup.py test

or simply

::

    $ make
